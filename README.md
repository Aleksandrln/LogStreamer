## Информация
Зависимости находятся в каталоге `requirements`
* `common` - основное 
* `test` - для запуска тестов

Путь к файлу лога задан в `settings.py` переменной `LOG_FILE_PATH`


## Запуск
* Установить зависимости:
 `pip install -r  requirements/common.txt`

* Выполнить `server.py`
  Например так:
 `python server.py`
